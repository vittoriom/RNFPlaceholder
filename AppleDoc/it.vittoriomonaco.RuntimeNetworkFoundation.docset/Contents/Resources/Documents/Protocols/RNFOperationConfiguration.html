name

The name of the operation

- (NSString *)name

Return Value

the name of the operation

Discussion

The name of the operation

This can be a simple name (e.g. ‘getTags’) or a string representation of a selector (e.g. ‘getItemsForTag:withCompletionBlock:’) In the first case, the operation will be retrieved through operationWithName: method of RNFEndpoint In the second case, the operation can be retrieved through operationWithName: but also by just calling the method on the RNFEndpoint
